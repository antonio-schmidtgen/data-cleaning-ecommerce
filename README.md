# Data Cleaning Project — E-commerce Transactions

## Objective
Prepare raw transactional data for analysis by cleaning and transforming it into a structured format.

## Dataset
Online Retail II dataset containing real-world e-commerce transactions.

## Process
- Inspected raw data
- Handled missing values
- Removed duplicates
- Filtered invalid data (negative quantities and prices)
- Converted data types
- Created new features (TotalPrice, Month, Year)

## Tools
- Python
- Pandas

## Result
Clean dataset ready for analysis and reporting.

## Business Relevance
Reflects real-world data cleaning tasks in e-commerce and data-driven companies.
