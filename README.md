# Data Cleaning Project — E-commerce Transactions

## Objective
Clean and transform raw transactional data into a structured format for analysis.

## Dataset
Online Retail II dataset containing real-world e-commerce transactions.

## Process
- Inspected raw data
- Handled missing values
- Removed duplicates
- Filtered invalid data (returns, incorrect prices)
- Converted data types
- Created new features (TotalPrice, Month, Year)

## Tools
- Python
- Pandas

## Result
Clean dataset ready for analysis and business reporting.

## Business Relevance
Simulates real-world data cleaning tasks commonly required in e-commerce and startup environments.
